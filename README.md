# Seikyu Companion

A fan-made companion app for Tales of Seikyu.

## Features

- Item database
- Villager gift guide
- Fish tracker
- Crop tracker
- Shrine tracker
- Upgrade calculator
- Animal and pet information
- Maps, shops, monsters, and schedules

## Progress Tracking

Progress is stored locally in your browser using local storage.

You can export and import your progress from the Backup page.

## Data Sources

Data has been compiled from:
- Tales of Seikyu game files
- Community wiki resources
- Publicly available community information
